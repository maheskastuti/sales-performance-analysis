# Sales Performance Analysis — Microsoft Excel

Excel-based sales analytics project with real datasets, formulas, charts, and dashboards.

## Sheets Included
- Sales Data — 36 orders across 12 months, 5 categories, 5 salespersons
- Monthly Analysis — Month-wise revenue, MoM growth %, cumulative revenue with bar and line charts
- Category Analysis — Revenue and units by category with % share breakdown
- Salesperson Analysis — Performance tracking with VLOOKUP demo and IF-based performance tags
- Dashboard — Live KPI cards for Total Revenue, Units Sold, Best Month, Top Category

## Excel Features Used
- Pivot-style summary tables
- VLOOKUP and IF formulas
- Cross-sheet formula references
- Bar charts and line charts
- MoM growth calculations
- Performance dashboards

## How to Open
Download the .xlsx file and open in Microsoft Excel or Google Sheets.
